## ❗ 本周易错点

### 1. 构造方法误解
- 写了有参构造后，默认无参构造不会再存在
- 调用构造方法必须匹配参数列表

### 2. this关键字
- this表示当前对象
- 用于区分成员变量和方法参数

### 3. 双指针错误
- fast负责遍历和筛选
- slow负责存放结果
- 错误：用slow做判断
