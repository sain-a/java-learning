# HelloSpringBoot 启动说明

## 1. 项目简介

本项目是第一个 Spring Boot Web 入门项目，实现了一个简单的 Hello 接口。

访问：

```text
http://localhost:8080/hello
````

返回：

```text
Hello Spring Boot
```

---

## 2. 技术栈

* Java
* Maven
* Spring Boot
* Spring Web

---

## 3. 项目结构

```text
HelloSpringBoot
├── pom.xml
└── src/main/java
    └── com.example.hellospringboot
        ├── HelloSpringBootApplication.java
        └── controller
            └── HelloController.java
```

---

## 4. 启动类

```java
@SpringBootApplication
public class HelloSpringBootApplication {
    public static void main(String[] args) {
        SpringApplication.run(HelloSpringBootApplication.class, args);
    }
}
```

说明：

* `main` 是程序入口
* `SpringApplication.run` 用来启动 Spring Boot 项目
* `@SpringBootApplication` 表示这是 Spring Boot 启动类

---

## 5. Hello接口

```java
@RestController
public class HelloController {

    @GetMapping("/hello")
    public String hello() {
        return "Hello Spring Boot";
    }
}
```

说明：

* `@RestController` 表示这是一个接口类
* `@GetMapping("/hello")` 表示访问 `/hello` 时执行该方法
* 返回值会直接显示在浏览器中

---

## 6. 自动配置理解

Spring Boot 会根据项目依赖自动配置常用组件。

例如引入 `spring-boot-starter-web` 后，Spring Boot 会自动配置：

* 内置 Tomcat
* Spring MVC
* Web 请求处理能力

所以开发者不需要手动配置服务器。

---

## 7. @SpringBootApplication 理解

`@SpringBootApplication` 可以理解为三个功能的组合：

* 配置类
* 自动配置
* 组件扫描

它会从启动类所在包开始，扫描当前包和子包中的组件。
