# 员工管理系统（控制台版）

## 📌 项目介绍

这是一个基于 Java 面向对象思想实现的简单员工管理系统，支持在控制台进行基本的员工管理操作。

## 🚀 功能

* 添加员工（全职 / 兼职）
* 删除员工
* 查询员工
* 修改员工信息
* 显示所有员工

## 🧱 技术点

* 面向对象编程（继承、多态、抽象类）
* 接口（Payable）
* 集合（ArrayList）
* 控制台输入（Scanner）

## 📂 项目结构

* Employee：抽象父类
* FullTimeEmployee / PartTimeEmployee：子类
* Payable：接口
* EmployeeService：业务接口
* EmployeeServiceImpl：实现类
* EmployeeSystemAPP：主程序入口

## 🧠 收获

通过该项目练习了：

* 类的设计与职责划分
* 多态在实际业务中的应用
* 简单的系统功能实现流程（增删查改）
